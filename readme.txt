Project Overview
A React-based Ice Cream Shop application with a Homepage, Flavors Page, and Login Page.
Built with reusable components, state management, and dynamic interactions.

Dependencies
- react-router-dom

Installation Commands
npx create-react-app sweet-scoop-app
cd sweet-scoop-app
npm install react-router-dom

How to Run
1. Unzip the file.
2. Run: npm install
3. Run: npm start
4. Open http://localhost:3000 in your browser.
